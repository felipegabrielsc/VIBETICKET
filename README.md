# VIBETICKET

Esse projeto na verdade é uma continuação do meu Trabalho de Conclusão de Curso feito na Fatec de Taquaritinga em 2025 meu grupo tinha decidido não dar continuidade no projeto porém na data atual eu decidi voltar a atualizar o projeto e melhorar ele, lembrando que o proejeto base original está no repositorio anexado abaixo tanto front-end quanto back-end

Front-end original: https://github.com/emSz345/VIBETICKET

Back-end original: https://github.com/emSz345/vibe-back


## Environment Variables

Ambos os arquivos .env nao estão disponiveis no repositorio porém disponibilizarei a base delas 

## Front-end

```dotenv
REACT_APP_API_URL=
MP_PUBLIC_KEY=
```

## Back-end

# BANCO DE DADOS

# NODE EMAIL
